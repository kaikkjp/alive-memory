# Deploy Shopkeeper - ONE COMMAND

Open your Mac terminal and run:

```bash
cd /path/to/shopkeeper
bash DEPLOY-NOW.sh
```

This will:
- ✅ Install everything on the VPS
- ✅ Build the frontend
- ✅ Set up Nginx
- ✅ Generate all SSH keys
- ✅ Print exact values for DNS and GitHub

Then just follow the printed instructions to:
1. Add DNS record (1 minute)
2. Add GitHub deploy key (1 minute)
3. Add GitHub secrets (2 minutes)

Total time: ~5 minutes

That's it. I did everything else.
